# Gmanman: Game Server Manager

## Features:

- Game Server Support:
  - Minecraft
  - Space Engineers
  - Ark
  - Factorio
  - Garry's Mod
- Start server button
  - stop server, disabled if numPlayers > 0, unless admin override
- Show status + player count
- Show logs
- Add/Disable Mods
  - with search
- create/load saves/backups
- Google Auth
